## 🛠️ Comment transformer un SVG en image Instagram

**Option rapide (sans logiciel) :**
1. Ouvrez le fichier `.svg` dans votre navigateur (double-clic ou glisser-déposer)
2. Faites une capture d'écran, ou utilisez un convertisseur en ligne SVG → PNG (ex: cloudconvert.com) en réglant la sortie sur 1080×1080px
3. Importez le PNG dans Instagram

**Option recommandée (plus propre) :**
1. Ouvrez le SVG dans **Figma** (gratuit) : créer un nouveau fichier → glisser le `.svg` dans le canvas
2. Modifiez les textes directement dans Figma (plus simple que dans le code)
3. Exportez en PNG 1080×1080 (clic droit → Export)

**Pour les carrousels :**
- Dupliquez `03-carousel-slide.svg` autant de fois que nécessaire (une slide par point)
- Changez le numéro (`1`, `2`, `3`...), le titre, le texte et la pagination (`1/4`, `2/4`...)

## 🔁 Workflow conseillé pour aller vite

1. Chaque dimanche, préparez les 3 posts de la semaine (légende + visuel)
2. Postez le matin entre 8h et 10h, ou le soir entre 18h et 20h (meilleurs horaires pour l'engagement BtoC)
3. Répondez aux commentaires/DM dans les 24h — ça compte pour l'algorithme et pour la conversion
4. Après 4 semaines, regardez les Insights : quels formats/sujets ont le mieux marché, et reproduisez ce qui fonctionne

## 🔗 Prochaines étapes possibles
- Créer une page `/liens` sur le site pour remplacer un Linktree externe (garde le trafic sur votre domaine)
- Décliner certains carrousels en **Reels** texte simple (vidéo avec les slides qui s'enchaînent + musique tendance)
- Une fois 10-15 posts publiés, activer la mise en avant payante (~5-10€) sur le post qui performe le mieux
