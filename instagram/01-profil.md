# Configuration du profil Instagram — Assur Emprunteur

## Nom du compte
`assur.emprunteur` (ou `assuremprunteur.fr` si le premier est pris)

## Nom affiché (30 car. max)
`Assur Emprunteur | Courtier`

## Catégorie professionnelle
Service financier / Courtier en assurance (à choisir dans les paramètres pro Instagram)

## Bio (150 car. max)

```
🏦 Courtier en assurance emprunteur
💰 Économisez jusqu'à 60% sur votre assurance prêt
✅ Loi Lemoine • Devis gratuit en 2 min
👇 Simulez ici
```

(4 lignes, ~140 caractères — ajustez le lien ci-dessous)

## Lien en bio
`https://assur-emprunteur.fr/#simulateur`

> Si vous voulez plus tard pousser plusieurs liens (simulateur, blog, contact, WhatsApp), on pourra créer une mini "Linktree maison" sur une page dédiée du site (`/liens`) pour garder 100% du trafic chez vous au lieu de dépendre d'un outil tiers.

## Photo de profil
Utilisez `logo-icone.svg` ou `logo-carre.svg` (déjà dans `public/logos`/`public/`) — format carré, fond uni, lisible en petit.

## Passer en compte professionnel
Paramètres → Compte → Passer à un compte professionnel → catégorie "Service financier" → relier une page Facebook si vous en avez une (optionnel mais recommandé pour les pubs futures).

## Activer les statistiques (Insights)
Indispensable pour suivre quels posts génèrent le plus de clics vers le simulateur.
